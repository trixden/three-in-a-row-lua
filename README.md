# Старт

Игра запускается через интерпретатор Lua

```shell
lua main.lua
```

# Команды

- `m x y d` - ход игрока
    - `x` - координата по оси X
    - `y` - координата по оси Y
    - `d` - направление движения (вдево `l`, вправо `r`, вверх `u`, вниз `d`)
- `c` или `q` - Выход

# Архитектура

- `command.lua` - парсит и обрабатывает введенные команды
- `model.lua` - модель поля игры
- `view.lua` - визуализация
- `vars.lua` - конфигурируемые переменные
